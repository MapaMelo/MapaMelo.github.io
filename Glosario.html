A continuación se encontraran distintas definiciones de conceptos, palabras y expresiones principales relacionadas con la asignatura de "Conceptos de redes y comunicaciones"


Protocolos

Reglas que permite que al cumplirlas se de el proceso de transmisión

Estos afectan la manera en la que se mandan e interpretan los mensajes



TCP

(Protocolo de Control de Transmisión, por sus siglas en inglés Transmission Control Protocol) es protocolo de red importante que permite que dos anfitriones (hosts) se conecten e intercambien flujos de datos. TCP garantiza la entrega de datos y paquetes (en-US) en el mismo orden en que se enviaron.


Esta se divide en las capas de:

  • Aplicación
  • Transporte
  • Internet
  • Acceso de red


OSI

El modelo OSI divide el sistema de comunicación de red en siete capas, que sirven para identificar los problemas de red.


Esta se divide en 7 capas o niveles:

  • 1. Física (Control de transmisión y conexión de los medios de comunicaciones. El protocolo principal es Ethernet, este organiza los datos que antes estaban en bloques de bits y se encarga en ponerlos en fila (los combinarte en paquetes con un encabezado (fila de bits) y con una trama final de control) La trama final de control, asegura que la traducción sea buena )
  • 2. Enlace de datos (se encarga de asegurar de la transformación)
  • 3. Red (direcciones y manejo de paquetes de datos)
  • 4. Transporte (se encarga de la segmentación de paquetes de datos)
  • 5. Sesión (asegurará que la comunicación en línea se mantenga (ej, control de línea en una videollamada))
  • 6. Presentación (Intermedio entre aplicación y sesión para asegurar que los datos que llegan basaron un sistema operativo sean entendibles por los que esta recibiendo cuando se tiene un dispositivo diferente con otro sistema operativo)
  • 7. Aplicación (lo que nosotros vemos, programas, manejos correos, http, SMTP, etc, https (más seguridad porque tiene información encriptada, cifrada))


Switch

“El puente”

Este conecta dispositivos como computadoras, impresoras, servidores, etc y genera una red (network) para compartir recursos (de un mismo espacio)



Router

Este es un dispositivo de hardware que sirve de punto de conexión entre una red local e Internet

Proporciona Wi-Fi y que generalmente está conectado a un módem.

Envía información desde Internet a los dispositivos personales, como computadoras, teléfonos o tablets



Dirección IP

Dirección de protocolo de internet

Series de números que identifica el dispositivo

Este ayuda a navegar en internet, comunicarse entre diferentes dispositivos e intercambiar datos

Usualmente tomamos una dirección pública de el proveedor de internet

Todo lo que se conecta a internet tiene una IP



Dominio

Es el nombre de “negocio” o página web en internet”

Un ejemplo de la estructura es:

  • www. (Nombre del domino). (Extensiones)


Internet

Este son redes interconectadas gracias a dispositivos conectados alrededor del mundo por redes conectadas de manera inalámbrica, o con cables de cobre o fibra óptica

Transmite información codificada en sistema binario

Es una gigantesca red de computadores interconectados



La Nube

"La nube" hace referencia a los servidores a los que se accede a través de Internet, y al software y bases de datos que se ejecutan en esos servidores

ELos datos se guardan en un centro de datos donde existen miles computadoras que guarda la información

No se necesita hardawe, memoria o espacio en disco



Sistema operativo

Es un programa para ejecución de tareas y funcionamiento de las aplicaciones

Este conecta dispositivos como computadoras, impresoras, servidores, etc y genera una red (network) para compartir recursos (de un mismo espacio)

Este organiza, almacena y da instrucciones a las aplicaciones para realizar tareas con eficiencia

Algunos ejemplos son:

  • Windows
  • Chrome OS
  • Android
  • iOS
